3D Printing Basics

What is 3D Printing?

Additive Manufacturing: Building objects layer by layer from the ground up, unlike traditional manufacturing that cuts away material.

Common Technology - FDM: Fused Deposition Modeling uses melted plastic extruded through a nozzle, like a hot glue gun that draws in 3D space.

üßµ Common Materials

PLA (Most Common)

Beginner Friendly
  • Biodegradable corn-based plastic
  • Easy to print, low warping
  • Prints at 190-220¬∞C
  • Great for classroom use

ABS

Advanced
  • Stronger and more flexible
  • Requires heated bed
  • Can warp easily
  • Fumes need ventilation

PETG

Intermediate
  • Strong and durable
  • Weather resistant
  • Food-safe options available
  • Slightly harder than PLA

Technology Constraints & Design Rules

‚ö†Ô∏è Critical Rules

Following these rules means the difference between successful prints and failures:

‚úì DO These Things

Rule Why It Matters Recommended Value
Minimum wall thickness Walls too thin will break or not print 2-3mm minimum
Design with flat base Ensures good bed adhesion Largest surface on bottom
Consider overhangs Gravity affects printing! Under 45¬∞ is safe
Check print orientation Affects strength and time Minimize overhangs
Keep first layer simple Prevents warping and lifting Wide, flat contact area

‚úó DON'T Do These Things

Avoid Problem Solution
Parts thinner than 1.5mm Will be fragile or won't print Make walls at least 2mm
Floating elements Nothing to print on = spaghetti Add supports or redesign
Tiny details under 3mm Printer can't resolve them Make details larger
Ignoring overhang angles Drooping, sagging prints Stay under 45¬∞ or add supports

üìê The Overhang Rule

0-45¬∞ ‚úì Safe

No supports needed. Prints beautifully.

Example: Gentle slopes, pyramids

45-70¬∞ ‚ö† Risky

May need supports depending on printer.

Example: Steep roofs, angled walls

70-90¬∞ ‚úó Impossible

Definitely needs supports or redesign.

Example: Horizontal bridges, overhangs

‚è±Ô∏è Print Time Factors

What affects print time:

  • Size: Bigger objects = more layers = longer time (exponential!)
  • Infill percentage: 10-20% is usually enough for classroom items
  • Layer height: 0.2mm is standard (lower = more detail = slower)
  • Print speed: 50mm/s is typical (faster = lower quality)
Rule of thumb: A 3-5cm object at normal settings prints in ~20 minutes

File Format

Must be "manifold" (watertight)

Scale

Verify size makes sense in real world!

üèÉ Quick Practice Exercise

1 Make a Simple Keychain Tag

This 5-minute exercise teaches all basic operations:

  1. Add a Box and resize to 20mm √ó 40mm √ó 3mm
  2. Add a Cylinder, make it 5mm diameter √ó 3mm height
  3. Position the cylinder at the top of the box
  4. Change the cylinder to a Hole (click "Solid" ‚Üí select "Hole")
  5. Select both objects and click Group to cut the hole
  6. Add Text, type your name, resize to fit
  7. Make text 2mm deep, position on the tag
  8. Group everything together
  9. Click Export ‚Üí STL
Congratulations! You just created your first 3D model ready for printing!
If you get lost or objects disappear, click the View Cube home button to reset your view!

Main Exercise: Custom Bookmark Clip

üìñ Project Overview

What: Personalized bookmark that clips onto book pages

Size: 30mm wide √ó 60mm tall √ó 2-3mm thick

Print Time: ~20 minutes

Difficulty: Beginner

üìã Step-by-Step Instructions

1

Create the Base Body (5 minutes)

  1. Open Tinkercad ‚Üí Create New Design
  2. Drag a Box onto the workplane
  3. Click the box and set dimensions:
    • Length: 30mm
    • Width: 60mm
    • Height: 2mm
  4. Optional: Round corners by placing small cylinder holes at corners
Straight corners work perfectly fine - skip rounding if it feels complicated!
2

Add the Clip Feature (5 minutes)

  1. Drag another Box onto workplane
  2. Set dimensions:
    • Length: 25mm (slightly narrower)
    • Width: 8mm
    • Height: 2mm
  3. Move this bar to the top of your bookmark
  4. Make it overlap the main body by 3-4mm
  5. Center it horizontally (use Align tool - press L)
  6. Select both pieces and click Group
The overlapping piece creates a flexible clip that grips the page!
3

Add Your Name (5 minutes)

  1. Click Text in the shapes panel (right side)
  2. Drag text onto your bookmark
  3. Click to edit and type your name or initials
  4. Adjust settings:
    • Height: 1-1.5mm (depth)
    • Font: Choose any (simple works best)
  5. Resize so letters are about 8-10mm tall
  6. Center the text on bookmark (use L to align)
  7. Press D to drop text onto surface

Choose your style:

  • Raised text: Leave as "Solid", then group with bookmark
  • Engraved text: Change to "Hole" to cut into bookmark, then group
4

Add Decoration (Optional - 5 minutes)

Personalize your bookmark with:

  • Simple shapes: Add a small star, heart, or subject icon
  • Math: Add œÄ symbol or numbers
  • Science: Small beaker (cylinder + cone)
  • Reading: Tiny book stack
  • Decorative holes: Small cylinders (3-4mm) as pattern
Keep decorations simple! Too many details = longer print time and harder to print.
5

Final Checks (3 minutes)

Verify your design:

  • Dimensions are roughly 30mm √ó 60mm √ó 2-4mm
  • Text is readable (big enough?)
  • Clip feature is clear at the top
  • No floating parts or disconnected pieces
  • Nothing thinner than 2mm

Rename project: Click "Tinker 1" at top ‚Üí Rename to "Bookmark_YourName"

6

Export for Printing (2 minutes)

  1. Click Export button (top right)
  2. Select .STL format
  3. File downloads as "YourProjectName.stl"
  4. Save to your USB drive or cloud storage
Your file is now ready to load into slicer software and print! üéâ

üö® Troubleshooting

Problem Solution
Text too small to read Make letters at least 8mm tall, use bold fonts
Clip bar too loose/tight Adjust overlap (3-5mm) in your design
Print time too long Reduce infill to 10% or make slightly smaller
Text not cutting through Increase text height to 1.5mm if using hole mode

Onshape Quick Demo

Why Onshape?

  • More powerful than Tinkercad - professional-grade CAD
  • Similar to SolidWorks, Fusion 360
  • Cloud-based, free for education
  • Better for precise measurements and engineering
  • Parametric design (edit history, make changes easily)

üÜö Tinkercad vs Onshape

Feature Tinkercad Onshape
Learning Curve Very easy (30 min) Moderate (3-5 hours)
Design Approach Building blocks Sketch-based
Best For Quick concepts, beginners, young students Precise parts, technical designs, measurements
Precision Good enough Extremely precise
Age Level Ages 8+ Ages 13+, adults
Price Free Free for education

üöÄ Quick Onshape Workflow

1

Sketch

Draw 2D profile/outline of your object

  • Use lines, circles, rectangles
  • Add dimensions (exact measurements)
  • Add constraints (perpendicular, parallel)
2

Extrude

Pull the 2D sketch into 3D space

  • Specify exact depth
  • Can add/remove material
  • Create complex shapes
3

Features

Add details and modifications

  • Fillets (rounded edges)
  • Chamfers (angled edges)
  • Holes, patterns, text
4

Export

Save as STL for 3D printing

‚ùì Frequently Asked Questions

How long to learn?

Tinkercad: 2-3 hours for basics

Onshape: 10-20 hours for proficiency

Print troubleshooting: Ongoing learning!

What age can students start?

Tinkercad: Ages 8+ with supervision

Physical printing: Ages 12+ (heat safety)

Onshape: Ages 13+ or mature students

Is it safe for classrooms?

PLA printing: Yes, low fumes

ABS printing: Needs ventilation

Hot parts: Adult supervision required

How much maintenance?

Weekly: Clean bed, check filament

Monthly: Lubricate rods, check belts

As needed: Replace nozzle, level bed

Workshop Preparation Checklist

‚ö° Quick Start Version

Minimum requirements to run this workshop:

  • Computer lab with internet access
  • 1 working 3D printer with filament
  • Sample prints to show
  • Printed handouts (Quick Reference)
  • Projector to demonstrate Tinkercad

üìÖ One Week Before

Technical Setup

  • Book computer lab with internet access
  • Verify all computers can access tinkercad.com
  • Check browsers are updated (Chrome/Firefox/Edge)
  • Install slicer software on demo computer (Cura or PrusaSlicer)
  • Create test Tinkercad account

3D Printer Preparation

  • Clean and calibrate 3D printer
  • Level the print bed
  • Test print a bookmark to verify settings
  • Load fresh filament (bright color recommended)
  • Prepare backup filament spool
  • Clean nozzle and check for clogs

Materials Gathering

  • Print 3-5 sample bookmarks (different designs)
  • Print examples of failed prints (for learning)
  • Collect other 3D printed examples
  • Prepare filament samples (different colors/types)

üìÖ 2-3 Days Before

Document Preparation

  • Print Quick Reference handouts (one per participant)
  • Print Bookmark Exercise Steps
  • Print extra copies (20% buffer)
  • Create sign-in sheet

Presentation Prep

  • Test screen sharing/projection
  • Bookmark tutorial videos (backup)
  • Prepare demo STL files on USB drive
  • Create slicer preset "Workshop Quick Print"

üìÖ Day Before

  • Visit room to plan layout
  • Verify projector works
  • Check power outlets for printer
  • Test WiFi signal strength
  • Charge backup devices
  • Pack extension cords if needed
  • Know IT support contact

üìÖ Day of Workshop - Arrive 30 Min Early

  • Arrange tables for screen visibility
