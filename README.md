**Files:** `workshop-2h-pt.html` | `workshop-2h-en.html`

**Learning Objectives:**
- Create functional 3D objects
- Understand complete print workflow
- Learn essential slicer settings
- Identify when to use supports
- Choose appropriate adhesion and quality settings

**Practical Exercises:**
1. **Part 1 (45 min):** Create a phone stand in Tinkercad

**Key Slicer Concepts Covered:**
- **Supports:** When and why they're needed (45Â° rule)
- **Bed Adhesion:** Brim vs Raft vs Skirt

**Workshop Structure:**
- 10 min: Setup and introduction
- 10 min: Complete 3D printing process overview
- 45 min: Tinkercad modeling
- 45 min: Slicer preparation with live demonstration
- 10 min: Conclusion and Q&A

## ğŸ“– How to Use These Materials

### For Instructors:
1. **Choose workshop duration** based on available time and participant experience
2. **Review preparation checklist** in each workshop file
3. **Follow structured timeline** with built-in checkpoints
4. **Use Design Principles** as reference material during workshop
5. **Adapt exercises** to your specific 3D printer and materials

3. **Follow exercises in order** - they build on each other
4. **Take your time** - listed durations are for group settings
5. **Experiment** after completing exercises

- Design Principles can be used as standalone reference
- Workshop exercises can be customized for specific topics
