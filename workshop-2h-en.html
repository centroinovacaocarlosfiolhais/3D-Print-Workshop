Exercise: Create a Phone Stand (35 min)

üí° Why a Phone Stand?
This project is ideal because:
  • It's a functional and useful object
  • Allows exploring angles and supports concepts
  • Has interesting design challenges
  • Introduces thinking about print orientation
  1. Create the base
    • Drag a "Box" to workspace
    • Adjust to create flat rectangular base
    • Give sufficient thickness for stability
  2. Create the back wall
    • Add another "Box" for the back wall
    • Position vertically at edge of base
    • Adjust height to accommodate a phone
    • Use rotation tool to create support angle (about 70-80¬∞)
  3. Add phone slot
    • Drag a "Box" and transform to "Hole"
    • Position and adjust to create slot where phone rests
    • Adjust depth and width
    • Group with base
  4. Create cable opening
    • Use "Cylinder" or "Box" as hole
    • Position at bottom to allow charging
    • Adjust size appropriate for USB cable
    • Group with rest of model
  5. Personalize with text or symbols
    • Add text with name or decorative icons
    • Try raised or recessed relief
    • Position on visible surfaces
  6. Final verification
    • Rotate object and check all angles
    • Confirm all parts are properly grouped
    • Check minimum thicknesses
    • Export as STL
‚ö†Ô∏è Important Points
  • Keep walls at least 2-3mm thick
  • Verify angle isn't too steep (may need supports)
  • Confirm base is wide enough for stability
  • Mentally test if real phone would fit

What is a Slicer? (5 min)

The Slicer is software that converts your 3D model into instructions the printer can follow. It "slices" the model into hundreds or thousands of thin layers and generates the path the printer nozzle should follow.

üì¶ Common Slicer Software
  • UltiMaker Cura: Free, easy to use, very popular
  • PrusaSlicer: Free, excellent for Prusa printers
  • Bambu Studio: For Bambu Lab printers

Essential Slicer Concepts (30 min)

1. Supports - 10 min

üèóÔ∏è What are Supports?

Supports are temporary structures the printer creates to hold parts that "float in air". They're like construction scaffolding - needed during printing but removed afterwards.

When to use supports:

  • Angles greater than 45¬∞: If a part has inclination over 45¬∞, it usually needs support
  • Overhangs: Parts projecting without anything underneath
  • Very long bridges: Horizontal distances without support

Practical demonstration:

  1. Show phone stand model
  2. Enable support visualization in Slicer
  3. Explain where they appear and why
  4. Rotate model to show different orientations
  5. Demonstrate how changing orientation can eliminate supports
üí° Design Tip
Whenever possible, design your objects to minimize or eliminate supports:
  • Think about orientation from the start
  • Use angles up to 45¬∞ when possible
  • Consider splitting complex objects into parts

2. Bed Adhesion - 10 min

üìê What is Bed Adhesion?

These are extra structures on the first layer that help the object adhere well to the print platform, preventing it from coming loose or moving during printing.

Adhesion types:

  • Brim: Extra border around first layer
    • Increases contact area
    • Easy to remove
    • Ideal for most objects
  • Raft: Complete platform under object
    • Maximum adhesion
    • Useful for objects with small base
    • Uses more material
    • Leaves mark on bottom surface
  • Skirt: Line around (doesn't touch object)
    • Only to prime the nozzle
    • Doesn't help with adhesion

Practical demonstration:

  1. Show each adhesion type in Slicer
  2. Explain when to use each
  3. Show physical examples of parts printed with each type
  4. Demonstrate how to remove brim vs raft

3. Print Quality (Layer Height) - 10 min

üìè What is Layer Height?

It's the thickness of each material layer. Thinner layers = more detail but more time. Thicker layers = faster but less detail.

Common profiles:

  • 0.1mm - High quality:
    • Very smooth surfaces
    • Fine details visible
    • Time: Can take 2-3x longer
    • Use for: Figures, decorative objects
  • 0.2mm - Normal quality:
    • Ideal balance
    • Good appearance and reasonable time
    • Use for: Most projects
  • 0.3mm - Draft:
    • Fast printing
    • Visible layers
    • Use for: Prototypes, functional parts where appearance doesn't matter

Practical demonstration:

  1. Show same object with different layer heights in Slicer
  2. Compare estimated print time
  3. Show physical examples of same part in different qualities
  4. Let participants feel the difference in surfaces
‚öñÔ∏è Choosing the Right Quality

Question to ask: "Is this decorative or functional?"

  • Decorative: Worth waiting for high quality
  • Functional: Normal or draft quality saves time
  • Prototypes: Always use draft first to test

Prepare for Printing (5 min)

  1. Select printer profile
  2. Choose material (PLA is most common)
  3. Set print quality (layer height)
  4. Enable supports if needed
  5. Choose adhesion type (usually brim)
  6. Click "Slice" to process
  7. Check estimated time and material amount
  8. View layer by layer (optional)
  9. Export G-code file to SD card or send to printer
