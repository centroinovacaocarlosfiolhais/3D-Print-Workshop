üé® Introduction to 3D Printing (10 min)

Basic Concepts

What is 3D printing?

3D printing builds objects layer by layer, like stacking very thin sheets of material until you form a solid object.

üí° Important 3D Design Principles
  • Thickness: All parts must have sufficient thickness (minimum 2-3mm)
  • Base: Objects need a stable base to prevent tipping
  • Supports: Suspended parts need support structures
  • Simplicity: Start simple and add complexity gradually

Quick Demonstration

Show physical print examples and explain:

  • How you can see the layers
  • Why some objects need supports
  • Examples of good and bad designs

Step by Step: Build the Keychain (30 min)

üí° Instructor Tip
Demonstrate each step on the projector and give 2-3 minutes for participants to replicate before moving to the next step.
  1. Create the base
    • Drag a "Box" shape from the sidebar to the workspace
    • Click on the shape to select it
    • Use the white handles in the corners to adjust size (make it flat like a card)
    • Adjust height to create a thin but strong base
  2. Make the keyring hole
    • Drag a "Cylinder" to the workspace
    • In the properties panel on the right, change from "Solid" to "Hole"
    • Adjust the cylinder size to create an appropriate hole
    • Position the cylinder at one end of the base
    • Use the height arrows to ensure it goes through the entire thickness
    • Select both shapes (Ctrl+A or drag a box around them) and click "Group"
  3. Add personalized text
    • Click "Text" in the sidebar
    • Drag the text onto the base
    • Double-click the text to edit (write name or word)
    • Adjust text size to fit nicely on the base
    • Decide if you want raised text (solid) or engraved text (hole)
    • Adjust height to create sufficient depth
  4. Personalize with decorative shapes
    • Choose shapes from the panel (stars, hearts, symbols, etc.)
    • Drag to the base
    • Decide if you want raised relief (solid) or recessed relief (hole)
    • Adjust size and position
    • Try rotating shapes using the rotation tool
  5. Finalize and verify
    • Select all final pieces (Ctrl+A)
    • Click "Group" to merge everything into a single object
    • Rotate the object to check from all angles
    • Confirm the hole goes all the way through
    • Verify all parts have adequate thickness
üéØ Checkpoints

During the exercise, circulate and verify:

  • Can everyone navigate the interface?
  • Is the hole working correctly?
  • Is the text legible and well positioned?
  • Are the shapes properly grouped?

üíæ Export for Printing (3 min)

  1. Click "Export" in the top right corner
  2. Select .STL format (standard format for 3D printing)
  3. Save the file to your computer
  4. Give it a descriptive filename (e.g., "keychain-name.stl")
üìÅ About STL Files

STL (STereoLithography) is the universal format for 3D printing. It stores your object's surface as a mesh of triangles that the printer can interpret.
