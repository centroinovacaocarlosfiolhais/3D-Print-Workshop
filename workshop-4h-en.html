üí° Onshape Philosophy

In Onshape you work in stages:

  1. Sketch: Draw the 2D profile
  2. Extrude: Give volume to drawing
  3. Modify: Refinements and details

Each stage stays in history and can be edited anytime!

Quick Exercise: Create a Cube (20 min)

Before starting the badge, let's practice the basic workflow:

  1. Create Sketch
    • Click "Sketch" icon in toolbar
    • Select "Top" plane (blue)
    • You're now in 2D drawing mode
  2. Draw Rectangle
    • Select "Rectangle" tool (R)
    • Click on center (origin) and drag
    • Release to create rectangle
  3. Add Dimensions (Constraints)
    • Select "Dimension" tool (D)
    • Click horizontal line, enter value (e.g., 50mm)
    • Click vertical line, enter value (e.g., 50mm)
    • Rectangle adjusts automatically
  4. Finish Sketch
    • Click green ‚úì in top right corner
    • Return to 3D view with your drawing on plane
  5. Extrude (Give Volume)
    • Click "Extrude" icon
    • Click rectangle you drew
    • Drag green arrow up or type height (e.g., 50mm)
    • Click green ‚úì
    • Now you have a 3D cube!
‚úÖ Concepts Learned
  • Sketch: 2D drawing on plane
  • Constraints: Dimensions that control drawing
  • Extrude: Transform 2D into 3D
  • Parametric: You can go back and change any value!

üè∑Ô∏è PART 1: Badge Base (45 min)

üéØ Goal of This Part
Create badge base shape with professional dimensions and rounded corners.

Design Planning (5 min)

Typical badge dimensions:

  • Width: 70-90mm
  • Height: 50-70mm
  • Thickness: 3-5mm (strong but not heavy)
üí≠ Thinking Like a Designer

Before starting, participants should decide:

  • Shape: Rectangular, oval, or custom?
  • Orientation: Horizontal or vertical?
  • Style: Minimalist or decorative?

Create Base Shape (40 min)

  1. New Sketch on Top Plane
    • Create new document if you don't have one open
    • Click "Sketch" ‚Üí Select "Top" plane
  2. Draw Centered Rectangle
    • "Rectangle" tool (R)
    • Click on origin (center) and drag
    • Or use "Center Rectangle" for easier centering
  3. Add Precise Dimensions
    • "Dimension" tool (D)
    • Set width: 80mm
    • Set height: 60mm
    • These dimensions are parametric - you can change later!
  4. Round Corners (2D Fillet)
    • "Fillet" tool in Sketch menu
    • Click a corner of rectangle
    • Set radius: 10mm (adjust to taste)
    • Repeat for all 4 corners
    • Rounded corners give professional look
  5. Finish Sketch
    • Verify profile is completely closed (green)
    • Click ‚úì to exit Sketch mode
  6. Extrude to Create Volume
    • Select "Extrude" tool
    • Click badge profile
    • Set thickness: 4mm
    • Verify it's in "New" mode (create new solid)
    • Click ‚úì
  7. Round 3D Edges (Optional but Recommended)
    • "Fillet" tool (3D, different from 2D)
    • Select top edges of badge
    • Set small radius: 0.5-1mm
    • This eliminates sharp corners and gives professional finish

‚úçÔ∏è PART 2: Text and Symbol Personalization (90 min)

Add Raised Text (40 min)

  1. Create New Sketch on Top Face
    • Click "Sketch"
    • Select top face of badge (will turn blue)
    • You're now drawing "on top" of badge
  2. Add Text
    • "Text" tool (T icon)
    • Click on face to position
    • Type desired name or text
    • Adjust font (choose clear and legible font)
    • Set appropriate size (start with 15-20mm height)
  3. Position Text
    • Use "Move" tool to adjust position
    • Or add distance constraints
    • Center horizontally if desired
    • Position vertically with space for logo
  4. Finish Text Sketch
    • Click ‚úì
    • Text appears as outline on surface
  5. Extrude Text in Relief
    • "Extrude" tool
    • Select text
    • IMPORTANT: Change from "New" to "Add"
    • Set relief height: 1-2mm
    • Click ‚úì
    • Text is now in 3D raised relief!
üí° Tip: Raised vs Recessed Relief
  • Raised Relief (Add): Text sticks out - more visible, more material
  • Recessed Relief (Remove): Engraved text - elegant, less prone to breaking
  • Try both and see which you prefer!

Add Logo or Symbol (50 min)

Option 1: Draw Simple Shapes

  1. Create new Sketch on top face
  2. Use drawing tools:
    • Circle: For circles
    • Polygon: For stars, hexagons, etc.
    • Line: For custom drawings
  3. Combine shapes to create unique symbol
  4. Extrude in Add or Remove

Option 2: Import SVG (More Advanced)

  1. Have SVG file of logo (can find on sites like flaticon.com)
  2. In Sketch, click "Insert" ‚Üí "DXF or DWG"
  3. Upload SVG file
  4. Adjust scale and position
  5. Extrude to create relief
üé® Personalization Ideas
  • Professional: Initials, position, company
  • Education: School, class, number
  • Events: Event name, date, logo
  • Hobby: Interest symbols (music, sports, etc.)
‚ö†Ô∏è Design Checkpoint

  • Is logo/symbol centered and balanced?
  • Is design not too busy?

üìé PART 3: Attachment System (30 min)

üîß Attachment Options

We'll create a functional attachment system. Choose one option:

Option A: Pressure Clip (Recommended)

  1. Sketch on Back Face
    • Rotate badge (or change view)
    • Create Sketch on back face
  2. Draw Clip Base
    • Small centered rectangle: 15mm x 8mm
    • Position in upper third of badge
  3. Extrude Base
    • Extrude in "Add"
    • Height: 3mm
  4. Create Clip Slot
    • New Sketch on top face of base
    • Rectangle: 12mm x 1.5mm (narrow slot)
    • Extrude in "Remove" going through entire base
  5. Round Edges
    • Fillet on sharp edges
    • Radius: 0.5mm
üí° How It Works

The slot allows inserting a clothes clip or commercial badge clip. Functional and simple!

Option B: Pin System

  1. Create Two Small Circles
    • Sketch on back face
    • Two circles: 3mm diameter each
    • Distance between centers: 15-20mm
  2. Extrude Holes
    • Extrude in "Remove"
    • Depth: 2mm (don't go all the way through)
  3. Use with Safety Pin
    • These holes allow attaching a safety pin
    • More permanent than clip

‚öôÔ∏è PART 4: Print Preparation (20 min)

Export from Onshape (5 min)

  1. Right-click "Part Studio" on left
  2. Select "Export"
  3. Format: STL
  4. Units: Millimeters
  5. Resolution: Fine
  6. Download file

Prepare in Slicer (15 min)

  1. Load in Slicer
    • Open Cura/PrusaSlicer
    • Import STL file
  2. Orientation
    • Badge should have main face up
    • Back face (with clip) down
    • This way text comes out with better quality
  3. Recommended Settings
    • Layer Height: 0.2mm (normal quality)
    • Supports: Probably not needed
    • Adhesion: Brim (recommended)
    • Infill: 20% is sufficient
  4. Slice and Verify
    • Click "Slice"
    • Check estimated time
    • Navigate through layers to verify
    • Pay special attention to text
‚úÖ Ready to Print!

Estimated print time: 1-2 hours

Material needed: 10-20g PLA
